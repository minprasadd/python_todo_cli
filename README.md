# Python_Todo_CLI

A command-line task management application built with Python featuring task creation, editing, priorities, due dates, and completion tracking.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Set task priorities
* Set due dates
* View all tasks

## Technologies Used

* Python
* File Handling
* Command Line Interface (CLI)

## How to Run

a. Clone the repository.
b. Open a terminal in the project folder.
c. Run the application:

```bash
python todo.py
```
